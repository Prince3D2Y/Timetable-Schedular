Setup steps:
Create your project folder (e.g., Python Project).

Inside that folder, create a subfolder.

Save app.py inside the project folder.

Save all three HTML files (index.html, add.html, bot.html) inside the subfolder folder.


Open a terminal/command prompt in the project folder and install Flask:
pip install flask


Run the command in terminal- cd "DIRECTORY"

Run the app with:
python app.py


Open your browser at http://localhost:5000.
